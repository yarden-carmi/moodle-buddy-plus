Moodle Buddy+
How to use
  1. Log into your university's Moodle.
  2. Open one of these pages:
    • The dashboard / course overview (URL ends in /my)
    • Any course page (URL contains /course)
    • An activity page (e.g. /mod/assign)
    • The Moodle video service (URL contains /videoservice)
  3. Click the Moodle Buddy+ icon in the browser toolbar.
Features
    Course page
  • One-click download of every resource in the course.
  • Detects new resources and activities (assignments, forums, …) since your last visit.
  • Download only what's new, or filter by files / folders.
  • Configurable folder structure and file-name format.
    • Dashboard
  • Scans every course on the overview page and surfaces updates per course.
  • Download a whole course (or just its new items) without leaving the dashboard.
  • Optional background scan that notifies you when courses change while you're logged in.
  • Group courses on disk by their dashboard category (FCL group).
  • When two courses share the same name in the same group, the registrar's course number is appended so each gets its own folder.
    • Resources
  • Folders are downloaded as a single zip per folder.
  • Moodle URL bookmarks are saved as .txt files containing the resolved external link.
    • Assignments & videos
  • Download assignment submission files, graded files, and feedback comments.
  • Download Moodle video service recordings and embedded Zoom recordings.
    • Other
  • Supports the Moodle Tiles course format.
Settings

Configure folder layout, file-name format, concurrency, custom selectors and more in the options page.

Feedback & contact

Open an issue on GitHub or email extensions@yardencarmi.com. If a feature breaks on your Moodle, please include the URL of the page where you saw the problem.

FAQ

Where do downloads end up?

Inside your browser's default download directory. The folder layout (per-course, per-section, grouped, "Moodle/" wrapper, …) is controlled in the options.

Two of my courses have the same name. Will the files mix?

No. When Moodle Buddy+ sees two courses with the same name in the same group, it appends each course's registrar number (e.g. Course Name (868320308)) to the folder name so they stay separate.

A Moodle "URL" resource — what gets downloaded?

A small .txt file containing the external link the URL activity points to. The extension does not try to scrape arbitrary external sites.

Resources keep showing up as "new" even though they aren't.

Clear the stored course data from the options. If it persists, please get in touch.

Can I limit concurrent downloads?

Yes — set the maximum in the options.

My dashboard has a lot of courses and the scan times out.

Reduce the number of courses shown on the dashboard, or cap the maximum scanned courses in the options.

Is Moodle Buddy+ open source?

Yes — the source is on GitHub.
