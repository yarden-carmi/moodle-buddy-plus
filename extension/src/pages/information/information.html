Moodle Buddy+
How to use?
  1. Log into your university's Moodle system
  2. Visit any of the following Moodle webpages:
    • Moodle Dashboard/Course Overview (URL ending on /my)
    • Any Moodle course page (URL includes /course)
    • Course activity pages (e.g. URL includes /mod/assign)
    • Moodle video page (URL includes /videoservice)
  3. Click on the Moodle Buddy+ logo in the extension bar
  4. Explore all the features Moodle Buddy+ has to offer

FAQ / Troubleshoot

The downloaded files are .php files and I can't open them. What is going on?

This usually occurs when a resource is pointing to a website. Please make sure that you can open the resource manually and it doesn't redirect you to another website. If this issue still occurs, please contact me.

Where are my files stored?

Files will always be downloaded to the default download directory of your browser. Depending on the download settings, the files will either be inside a folder with the same name as the course name (default), a special "Moodle" folder or directly in the download directory.

Can I limit the maximum amount of concurrent downloads?

Yes! You can configure this in the settings.

Random resources are being shown as new although they aren't. What is going on?

Please clear the stored course data in the settings. This should hopefully fix it. If this issue still occurs, please contact me.

There are too many courses in my dashboard and the scanning fails. What can I do?

You can either customize your Moodle dashboard to limit how many courses are displayed or configure the maximum amount of scanned courses in the settings.
